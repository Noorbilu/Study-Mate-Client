StudyMate is a simple web application built using the MERN Stack (MongoDB, Express, React, Node.js).
The main goal is to help students find study partners easily.
It lets users:

Sign up / Log in securely.
Create a profile describing what they want to study.
Search and filter a list of available partners.
Send connection requests to study together.
